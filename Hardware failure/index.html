Сбой оборудования

- количество часов на выполнения задания

- количество часов на конвертирования в нужный формат

- количество форматов

- вероятность ошибки или сбоя

Результат

- количество серверов нужное для конвертации

Задача должна быть реализована на языке JavaScript. Представьте, что компания предоставляет сервис по доставке видео контента на мобильные устройства в различных форматах. По условиям контракта с поставщиками видео должно быть доступно клиентам не позднее чем через m часов, после того как Ваша компания получает мастер-копию. Конвертирование мастер-копии в нужный формат занимает n часов, где n < m. Вероятность ошибки или отказа оборудования - x%. Конвертирование занимает все ресурсы выделенного для этого сервера. Серверов может быть сколь угодно много. Количество форматов обозначается буквой с. Необходимо составить программу, которая принимала бы на вход m, n, x, c и выдавала бы минимальное количество серверов, необходимое чтобы для каждого формата вероятность после m часов не получить результат была бы 1% или менее.
